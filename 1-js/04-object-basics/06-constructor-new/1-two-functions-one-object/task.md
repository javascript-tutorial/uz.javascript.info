# Ikki funktsiya - bitta obyekt

`A` va `B` funktsiyalarini `new A()==new B()` kabi yaratish mumkinmi?

```js no-beautify
function A() { ... }
function B() { ... }

let a = new A;
let b = new B;

alert( a == b ); // true
```

Agar shunday bo'lsa, unda ularning kodlariga misol keltiring.
