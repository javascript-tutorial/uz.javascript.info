# Noyob qator a'zolarini filtrlang

Keling, `arr` massiv bo'lsin.

`arr` ning noyob elementlari bo'lgan massivni qaytarishi kerak bo'lgan `unique(arr)` funktsiyasini yarating.

Masalan:

```js
function unique(arr) {
  /* sizning kodingiz */
}

let strings = [
  "Hare",
  "Krishna",
  "Hare",
  "Krishna",
  "Krishna",
  "Krishna",
  "Hare",
  "Hare",
  ":-O",
];

alert(unique(strings)); // Hare, Krishna, :-O
```
