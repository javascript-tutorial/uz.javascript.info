# Elementni tozalang

Elementdan hamma narsani olib tashlaydigan “clear(elem)” funksiyasini yarating.

```html run height=60
<ol id="elem">
  <li>Hello</li>
  <li>World</li>
</ol>

<script>
  function clear(elem) {
    /* sizning kodingiz */
  }

  clear(elem); // ro'yxatni tozalash
</script>
```
