# O'tkazish paneli kengligi qancha?

Standart aylantirish panelining kengligini qaytaradigan kodni yozing.

Windows uchun u odatda `12px` va `20px` orasida o ªzgarib turadi. Brauzer buning uchun joy ajratmasa (aylantirish paneli matn ustida yarim shaffof bo'ladi, bu ham sodir bo'ladi), u `0px` bo'lishi mumkin.

P.S. Kod har qanday HTML hujjati uchun ishlashi kerak, uning mazmuniga bog'liq emas.
